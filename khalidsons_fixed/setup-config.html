⚙️ Supabase Config Setup

Ek baar setup karo — phir sab kuch kaam karega

1 Supabase Dashboard kholein

Supabase Dashboard Open Karein →

2 Project Select karein → Settings → API

Wahan se copy karein:

  • Project URL — jaise https://xxxx.supabase.co
  • anon public key — lamba JWT token
3 Neeche paste karein aur Save karein
4 Email Confirmation Band Karo (Important!)

Supabase Dashboard → Authentication → Providers → Email

"Confirm email" toggle → OFF kar do → Save

Yeh karne se signup ke baad seedha login ho sakta hai bina email verify kiye.
